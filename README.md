# BTE Platform

## BaZi Technology Engine

BTE (BaZi Technology Engine) là nền tảng phân tích Bát Tự thế hệ mới, được xây dựng với mục tiêu chuẩn hóa tri thức Bát Tự theo hướng dữ liệu, thuật toán và AI.

## Mục tiêu

- Chuẩn hóa toàn bộ dữ liệu Can Chi.
- Chuẩn hóa Tàng Can.
- Chuẩn hóa Thập Thần.
- Xây dựng Rule Engine.
- Phân tích Thân Vượng / Thân Nhược.
- Xác định Dụng Thần.
- Tính Đại Vận.
- AI giải thích toàn bộ kết quả.
