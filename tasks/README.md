# BTE Platform Task Management

Version: 1.0

---

# Purpose

Thư mục `tasks/` quản lý toàn bộ công việc phát triển của BTE Platform.

Đây là nguồn quản lý chính thức cho:

- Công việc đang thực hiện
- Công việc tương lai
- Roadmap
- Release
- Refactor
- Bug Fix
- Testing

---

# Folder Structure

roadmap/

Lộ trình phát triển.

current/

Các Task đang thực hiện.

backlog/

Các Task sẽ làm.

completed/

Task đã hoàn thành.

engines/

Task theo từng Engine.

database/

Task về Database.

testing/

Task về Testing.

release/

Checklist phát hành.
