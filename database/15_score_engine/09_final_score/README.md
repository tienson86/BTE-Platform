09_FINAL_SCORE

Mục đích
--------
Chuẩn hóa kết quả cuối cùng của Score Engine.

Các file
--------

01_grade.csv
Quy đổi tổng điểm sang cấp độ (S+, S, A+, A...).

02_rating.csv
Đánh giá từng khía cạnh bằng thang sao.

03_confidence.csv
Đánh giá độ tin cậy của kết quả.

04_dimension_weight.csv
Quy định trọng số của từng module trong Score Engine.

05_recommendation.csv
Sinh khuyến nghị theo tổng điểm.

06_output_mapping.csv
Ánh xạ dữ liệu từ các module sang kết quả đầu ra.
