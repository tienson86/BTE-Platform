# am_duong

## Giới thiệu

Đây là Master Calendar của nền tảng BTE.

Module chịu trách nhiệm chuyển đổi giữa:

- Dương lịch
- Âm lịch
- Can Chi
- Tiết khí
- Tháng Bát Tự

## Đầu vào

- Ngày sinh
- Giờ sinh
- Múi giờ

## Đầu ra

- Âm lịch
- Can Chi năm
- Can Chi tháng
- Can Chi ngày
- Can Chi giờ
- Tiết khí
- Tháng Bát Tự
