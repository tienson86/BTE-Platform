# Module 09 - Hôn nhân

## Mục tiêu

Đánh giá toàn diện chất lượng hôn nhân trong lá số Bát Tự dựa trên nhiều yếu tố:

- Phối Ngẫu Tinh
- Cung Phu Thê
- Thập Thần
- Thân Vượng / Thân Nhược
- Dụng Thần
- Hỷ Thần
- Kỵ Thần
- Thần Sát
- Quan hệ Can Chi
- Xung Hợp Hình Hại

Module này KHÔNG tự tính toán.

Module chỉ đọc dữ liệu đã được tính từ các module khác.
